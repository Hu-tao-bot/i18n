# Interaction
[THAI](../lang/interaction/README.md) | ENG
----
Interaction is folder for show Embed like Character info, Summary world, Error and etc.

## Keys
```
- info: About Hu tao bot
- login: Check login status.
- profile: Profile data.
- help: Help commands.
- realtime:Realtime data of Resin, Daily Quest, Survey, Realm Currency and etc.
- redeemcode: Redemption code.
- summary:
    - info: Get summary (days used, achievement, characters, etc.)
    - world: Get world explorations summary.
    - teapot: Get serenitea pot summary.
- abyss:
    - info: Get Spiral Abyss in this time.
    - floor: Get Spiral Abyss with floor.
- characters:
    - list: Get character list.
    - info: Get character info.
- wish:
    - list: Get info wish banner.
- daily:
    - auto-cliam-opened: Enable auto daily login reward.
    - auto-cliam-closed: Disable & Disable auto daily login reward.
    - daily-month: Get daily login reward of this month.
    - daily-cliamed: Got reward of daily login completed.
```

## Keys of errors
```
- LOGIN_REQUIRED: Login required.
- USER_NOT_PUBLIC: User not be enable this feature.
- DATA_NOT_FOUND: Not found data.
- ABYSS_FLOOR_USER_NOT_FOUND: Not found floor of abyss
- ABYSS_FLOOR_RANGE_ERROR: Invalid floor of abyss.
- CHARACTERS_NOT_FOUND: Character not found.
- RATE_LIMIT: Rate limit use commands.
- REDEEM_INVALID: Redemption not complete.
- REDEEM_CLAIMED: Redemption already claimed.
- REDEEM_COOLDOWN: Redemption has rate limit.
- REDEEM_EXCEPTION: AR. lower than 10.
- REDEEM_LOGIN_REQUIRED: User not enable redemption code.
- COMMAND_NOT_SUPPORT: Commmand doesn't support with login UID and Hoyolab ID
- GACHA_NOT_FOUND: Not found wish banner this version.
- DAILY_ALREADY_CLAIMED: Daily already claimed.
- UNKNOW_ERROR: Unknown error.
```
